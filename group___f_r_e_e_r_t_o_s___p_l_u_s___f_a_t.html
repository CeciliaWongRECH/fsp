Detailed Description

Middleware for the Fat File System control on RA MCUs.

Overview

The FreeRTOS Plus FAT performs Fat File System control. This middleware is based on open source. Please refer to the following URL for details. https://www.freertos.org/FreeRTOS-Plus/FreeRTOS_Plus_FAT/index.html

Features

The FreeRTOS Plus FAT module supports the following features:

  • File read support
  • File write support

Usage Notes

RM_FREERTOS_PLUS_FAT_Open is an initialization function. Please use it before using the FreeRTOS Plus FAT functions. The main operations of the initialization function are as follows.

  1. Allocate an FFDisk_t Structure.
  2. Mount a Partition.
  3. Add a Partition to the Virtual File System.

Examples

FreeRTOS Plus FAT Basic Example

This is a basic example of minimal use of the FreeRTOS Plus FAT in an application.

void freertos_plus_fat_example (void)
{
size_t file_size;
int close_err;
fsp_err_t err;
size_t test_xCount;
FF_FILE *p_file;
usb_pin_setting(); /* USB pin function and port mode setting. */
/* Open Device Driver.*/
err = RM_FREERTOS_PLUS_FAT_Open(&g_freertos_plus_fat0_ctrl, &g_freertos_plus_fat0_cfg);
handle_error(err);
/* Check USB device attached and mounted. */
while(0 == gp_cfg.pxDisk->xStatus.bIsMounted){
vTaskDelay(1);
}
/* Open the source file in read and write mode. */
p_file = ff_fopen((const char *) FREERTOS_PLUS_FAT_EXAMPLE_FILE_NAME, "w");
if (0 != p_file)
{
/* Write however many bytes were read from the source file into the destination file. */
file_size = ff_fwrite( g_file_data, sizeof(g_file_data), 1, p_file );
close_err = ff_fclose( p_file );
}
/* Open the source file in read mode. */
pxSourceFile = ff_fopen((const char *) FREERTOS_PLUS_FAT_EXAMPLE_FILE_NAME, "r");
if(0 != p_file)
{
/* Read sizeof( Buffer ) bytes from the source file into a buffer. */
test_xCount = ff_fread(read_Buffer, sizeof(g_file_data), 1, p_file );
close_err = ff_fclose( pxSourceFile );
}
err = RM_FREERTOS_PLUS_FAT_Close(&g_freertos_plus_fat0_ctrl);
handle_error(err);
}
