Motor Interface
Interfaces

Detailed Description

Interface for Motor functions.

Summary

The Motor interface provides Motor functionality.

Implemented by:

  • Motor Sensorless Vector Control (rm_motor_sensorless)
  • Motor encoder vector control (rm_motor_encoder)
